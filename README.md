# FancyBashV1
Make your Bash Terminal Fancier!

This script backups your default ".bashrc" config and setups a better and cleaner one.

Additional functionality:
- Timestamps
- Command counter
- Blinking "$"

Usage:
- git clone https://github.com/Sejsig/FancyBashV1
- cd FancyBashV1
- chmod +x fancybash.sh
- ./fancybash.sh
- *Restart your terminal*
